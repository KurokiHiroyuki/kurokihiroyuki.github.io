Shopee Maintenance Tool

Shopee 出品メンテナンス支援ツール / Listing maintenance helper

概要 / Overview

本ツールは、Shopee Open Platform API (v2) を利用して、自身が運営するショップの 出品商品情報を取得・整理し、カテゴリやSEOタイトルの改善案を作成・反映するための 個人用ユーティリティです。

This is a private utility that uses the Shopee Open Platform API (v2) to fetch and organize the operator's own shop listings, and to prepare and apply improvements to product categories and SEO titles.

主な機能 / Features

  • OAuth によるショップ連携とアクセストークンの自動更新
  • 出品商品の一覧・詳細の取得とローカル保存（SQLite）
  • 商品カテゴリ・SEOタイトルの最適化案の生成

API の利用範囲 / API usage

取得するのは運営者自身のショップデータのみで、商品情報の参照・更新に限られます。 認証情報はサーバーやリポジトリに保存・公開されません。

The tool accesses only the operator's own shop data, limited to reading and updating product information. Credentials are never stored in or published to any repository.
